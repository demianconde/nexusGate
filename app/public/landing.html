Integração em 1 minuto

Você não muda sua stack. O NexusGate é compatível com o formato da API da OpenAI: troque a base_url e use a sua chave do NexusGate. O resto continua igual.

1

Cadastre suas chaves BYOK

Adicione suas API keys (OpenAI, Anthropic, Qwen) no cofre cifrado. Login do painel via Supabase.

2

Aponte a base URL

Use https://api.nexusgate.com.br/v1 e sua chave nxg_... no header.

3

Roteamento + cache automáticos

Interceptamos, escolhemos o modelo certo, servimos do cache quando dá e devolvemos em streaming.

exemplo.py⧉
from openai import OpenAI

client = OpenAI(
    base_url="https://api.nexusgate.com.br/v1",
    api_key="nxg_a1b2c3d4.•••••••",  # chave do NexusGate
)

resp = client.chat.completions.create(
    model="nexus-auto",      # roteamento automático
    messages=[{"role": "user",
               "content": "Escreva testes para X"}],
    stream=True,
)
